Capstone Project Presentation

Challenges Along the Way

  • Figuring out how to structure pages correctly
  • Making the layout responsive across devices
  • Aligning images and text with Bootstrap
  • Fixing CSS conflicts and spacing issues
  • Debugging broken links and formatting problems

Explanation of the Process

I started with simple wireframes and rough drafts of each page. From there, I built the structure using HTML and gradually added Bootstrap components for layout and design. After that, I focused on interactivity and style — linking CSS files, fixing spacing, and making sure everything worked across different screen sizes. I frequently tested and tweaked my code while referring to documentation and examples.

Summary of My Learning

This project taught me how to apply what I’ve learned in class to a real project. I got better at problem-solving, especially when things didn’t work as expected. I learned to debug my code, read Bootstrap documentation more carefully, and stay organized with my files. Most importantly, I became more confident in my web development skills and proud of what I was able to create on my own.
